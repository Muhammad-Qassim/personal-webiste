Intro

This is the personal website of Muhammad Qassim. I am currently studying computer science at Texas State University. I absolutely love learning about new technology and how they work, and I hope to one day use my knowledge to help others in some way. Outside of school, I enjoy spending time with my friends and family, going on hikes, and playing video games.

I really enjoy coding and am proficient in several languages, including C++, Python, Go, HTML and CSS. I find it to be a very rewarding experience when I can take a complex problem and break it down into manageable pieces of code that work together to solve the issue. It's also gratifying to see the end result of my efforts when everything comes together and works as intended.

I'm currently working on an ARM superscalar pipeline simulator. The simulation includes handling hazard detection and mitigation, fully associative cache operation, and arbitrary memory location load/store. I'm using the GO programming language for this project.
